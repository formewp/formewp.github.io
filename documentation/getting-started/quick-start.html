# Quick Start

Know what you're doing and are on Unix? Simply make sure you have all the prerequisites in place, then follow the base install instructions below, and create your plugin or theme.

Need more detail or use Windows? We've got more in depth explanations on how to set up your local environment, base WordPress installation and plugins/themes.

# Base Install With Valet

# Prerequisites

  • mac/linux
  • Valet (opens new window)
  • php >= 8.0 or above on cli and server
  • wp cli (opens new window)
  • node 16
  • curl/wget
  • composer ^2
  • bash
  • forme codegen ^1.3 (opens new window)

# Steps

  1. Set up a new mysql database and make a note of the credentials

  2. cd to your sites directory. E.g.

cd ~/sites
  1. Install and configure a new WordPress site directory (you'll be asked for the db credentials)
forme base new hello-world
  1. cd into this new site directory
cd hello-world
  1. Add the valet driver if you haven't already
cp utils/FormeServerValetDriver.php ~/.config/valet/Drivers/FormeServerValetDriver.php
  1. Enable https
valet secure
  1. Visit the site e.g. https://hello-world.test in your browser and set up the admin user

# Base Install With Docker

# Prerequisites

  • mac/linux
  • Docker & docker-compose - you can just install Docker Desktop (opens new window)
  • php >= 8.0 or above on cli
  • wp cli (opens new window)
  • node 16
  • curl/wget
  • composer ^2
  • bash
  • forme codegen ^1.3 (opens new window)

# Steps

  1. cd to your sites directory. E.g.
cd ~/sites
  1. Install and configure a new WordPress site directory
forme base new hello-world
  1. cd into this new site directory
cd hello-world
  1. Initialise docker configuration
forme ketch init
  1. Launch docker
forme ketch up
  1. Visit the site at http://localhost in your browser and set up the admin user

# New Plugin / Theme

# Prerequisites

  • mac/linux
  • php >= 8.0 or above on cli
  • forme codegen ^1.3 (opens new window)
  • node 16
  • git
  • composer ^2
  • bash
  • git cliff (opens new window) (optional)

# Steps

  1. cd into your repos directory. e.g.
cd ~/repos`
  1. Create a new plugin or theme
# plugin
forme new plugin hello-world

# theme
forme new theme hello-world
  1. cd to the base wordpress installation.
cd ~/sites/hello-world
  1. Symlink the repo directory into your base WordPress site installation or link it to your docker
# base installation e.g. valet
forme base link ~/repos/hello-world-plugin/

# docker
forme ketch link ~/repos/hello-world-theme/
# you might also need to restart container if it's running
forme ketch restart
  1. Update php dependencies
composer update
